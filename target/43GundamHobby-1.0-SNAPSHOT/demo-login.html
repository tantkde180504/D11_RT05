📋 Flow hoạt động
  1. Nhập thông tin đăng nhập
    • Email và password
    • Validation form
  2. Gửi request đến API
    • POST /api/login
    • Kiểm tra database
  3. Nhận response
    • JSON: {success, role, fullName}
    • Lưu vào localStorage
  4. Hiển thị thông báo
    • 🎉 Chào mừng + tên + role
    • 🚀 Đang chuyển trang...
  5. Chuyển trang theo role
    • ADMIN → dashboard.jsp
    • STAFF → staffsc.jsp
    • CUSTOMER → index.jsp
